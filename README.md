## 快速开始

按照以下步骤运行项目：

1. **克隆仓库：**

   使用以下命令将仓库克隆到本地：

   ```bash
   git clone https://github.com/edwin986/Elog.git
   ```

2. **打开文件：**

   在浏览器中打开 `index.html` 文件以查看页面效果。

3. **（可选）编辑代码：**

   您可以根据需要修改 HTML 文件并刷新浏览器以查看更改。
